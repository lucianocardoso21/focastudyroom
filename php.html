Formatação de Strings

Abaixo temos uma lista de comandos usados para formatação de strings:

  1. printf(): permite exibir uma string já formatada aceita vários parametros de formatação (ver manual). ex: 
                        
                        $prod = "leite"; // string
                        $preco = 4.5; //float
                        printf("O %s está custando R$ %.2f", $prod, $preco); // %s - string %f - float
  2. print_r(): Exibe coleções, objetos e variáveis compostas (vetores e matrizes) de maneira organizada. ex: 
                        
                        $x[0]=3;
                        $x[1]=12;
                        $x[2]=9;
                        print_r($x);
                        $v = array (3,7,6,2,1,9)
                        orint_r($v);
                        // tb pode ser substituido por var_dump(); ou var_export();
  3. wordwrap(): Cria quebras de linha ou divisões em uma string em um tamanho especificado, tanto de forma interna como de forma externa. ex: 
                        
                        $s = "string gigaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaante...."
                        $res = wordwrap($txt, 10, "<br>\n", false);
  4. strlen(): Permite verificar o tamanho de uma string, contando seus caracteres (inclusive espaços em branco).

  5. trim(): Elimina espaços em branco antes e depois de uma string.

  6. ltrim(): Elimina espaços no início de uma string. left trim

  7. rtrim(): Elimina espaços em branco no final de uma string. right trim

  8. str_word_count(): Conta quantas palavras uma string possui e armazena cada palavra em uma posição de array. ex: 
                        
                        $frase = "eu vou estudar php";
                        $count = str_word_count($frase, 1); //aceita parametros 0 (ou sem nada), 1 e 2
                        print_r($count); //print_r para mostrar um vetor.
    O param 0 (ou sem nada) mostra a quantidade de palavras. Param 1 mostra o array e o param 2 mostra o array na posição em que cada palavra começa.

  9. explode(): Quebra uma string e coloca os itens em um vetor. ex: 
                        
                        $frase = "Curso em Vídeo";
                        $vetor = explode(" ", $frase); //quebra no param selecionado. nesse caso, quebra onde tiver espaço (" ").
                        print_r($vetor);
  10. str_split(): Coloca cada letra de uma string em uma posição de um vetor. Ex: 
                        
                        $nome = "Maria";
                        $vetor = str_split($nome);
                        print_r($vetor);
  11. implode(): Transforma um vetor inteiro em uma string. Ex: 
                        
                        $arr[0] = 'Hello';
                        $arr[1] ='World!';
                        $arr[2] ='Beautiful';
                        $arr[3] ='Day!';
                        $texto = implode(" ",$arr);
                        echo implode(" ",$arr);
                        print ($texto);
  12. chr() : Retorna um caractere de acordo com seu código ASCII passado como parâmetro.
  13. ord(): Retorna o código ASCII de um caractere passado como parâmetro.
  14. strtolower() = passa a string toda para minusculas.
  15. strtoupper() = passa a string toda para maiusculas.
  16. ucfirst() = passa a primeira letra da string para maiusculas. Apenas a 1a. Pode ser usada em conjunto com outras funções, por exemplo: 
                        
                        $nome = 'lUCiANo';
                        $nome2 = strtolower($nome); // resultado: luciano
                        $nome3 = ucfirst(@nome2); // resultado: Luciano
  17. ucwords() = passa a primeira letra de cada palavra da string para maiusculas.
  18. strrev() = mostra a string revertida.
  19. strpos() = mostra em que posição a string procurada foi encontrada. A str tem que estar escrita EXATAMENTE da maneira pesquisada. Exemplo: 
                        
                        $frase = 'Estou aprendendo PHP';
                        $pos = strpos($frase, 'PHP'); // retorna o valor 17, posição onde aparece. Ele não acha se php estiver
                        escrito diferente (em minusculo por exemplo).
  20. stripos() = mostra em que posição a string procurada foi encontrada. Ignora formatação Exemplo: 
    
                    $frase = 'Estou aprendendo PHP';
                    $pos = strpos($frase, 'php'); // retorna o valor 17, posição onde aparece
  21. substr_count() = conta quantas vezes a substring aparece. Exemplo: 
    
                    $frase = "Estou aprendendo PHP no curso de PHP";
                    $count = substr_count($frase, "PHP");
                    print("$count"); // retorno: 2
  22. substr() = Fatia a string. Exemplo: 
    
                    $site = "Curso em Vídeo";
                    $site2 = substr($site,0,5) // fatia a str $site começando na pos 0 e andando 5 posições.
                    $site3 = substr($site,6) // fatia a str $site começando na pos 6 até o final.
                    $site4 = substr($site,-5,2) // fatia a str $site começando de tras para frente, andando 2 pra frente.
                    print("$site2"); // retorno: Curso
                    print("$site3"); // retorno: em Vídeo
                    print("$site4"); // retorno: Vi
  23. str_pad() = faz a string caber em um determinado espaço. Sintaxe: str_pad($var,30," ",STR_PAD_RIGHT); onde $var é o nome da variável, 30 é a quantidade de caracteres a serem ocupados, " " é com o que os espaços serão preenchidos (nesse caso, com espaços em branco) e STR_PAD_RIGHT é como os espaços serão preenchidos. STR_PAD_RIGHT - direita STR_PAD_LEFT - esquerda STR_PAD_CENTER - esq e dir.

  24. str_repeat() = repete o texto informado. Ex: 
    
                    $txt = str_repeat("Php",5);
                    print("$txt"); // PhpPhpPhpPhpPhp
  25. str_replace() = Substitui uma string. Segue a mesma lógica do strpos()/stripos(), podendo ser 
    
                    str_ireplace(). Ex:
                    $frase = "gosto de matematica";
                    $nova = str_ireplace("matematica", "php", $frase);
                    echo $nova // gosto de php
