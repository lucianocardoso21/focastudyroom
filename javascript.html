Criando variáveis em JavaScript


1- Textos (string):

Pode-se usar aspas duplas (" ") ou simples (' '). Ex:

var nome = "Luciano"; // Cuidado: nomes de var sao case sensitive


2- Numéricos (int ou float):

Int são variáveis do tipo INTEIRO, ou seja, não possuem a parte decimal. Já float são variáveis do tipo DECIMAL, aceitam números com vírgula. Ex:

var n1 = 3.45; (float) // posso atribuir diretamente o valor da var ou
n1 = 2; (int) // declarar a var e depois atribuir um valor a ela


3- Booleanos:

Exibem valores lógicos true e false.

let situacao = false;
var aprovacao = true;


Diferenças entre CONST, VAR e LET:

Enquanto as variáveis declaradas com let têm escopo de bloco, instrução ou expressão, as variáveis declaradas com var têm escopo global ou escopo de função. Antes de tudo é necessário entender o que é escopo de variáveis.
Escopo de variáveis nada mais é do que onde uma variável existe. Faça-se a seguinte pergunta: se eu declarar uma variável dentro de um bloco de if por exemplo, esta variável deve existir fora deste bloco de if? Se a sua resposta foi sim, você deve usar var para declará-la. Mas se a sua resposta foi não, ou seja, esta variável deve existir  apenas dentro do if , você deve usar let para declará-la neste caso. A ideia funciona para qualquer tipo de bloco: if, for, while, função, etc.
Já a variável const é um tipo de variável constante, ou seja, nunca não pode ser alterada (nunca diga nunca🤫).


Inserindo e recuperando dados

Inserindo dados:

var usuario = prompt("Insira o nome do usuario:"); // insere dados via prompt

Recuperando dados:

alert(variavel); - solta uma msg numa janela popup
document.write(variavel); - escreve na página
console.log(variavel); - escreve no console.log


Operadores em JavaScript


1- Aritmericos

  • Soma: console.log(n1 + n2); Retorno: 17
  • Subtração: console.log(n1 - n2); Retorno: 3
  • Multiplicação: console.log(n1 * n2); Retorno: 70
  • Divisão: console.log(n1 / n2); Retorno: 1.4285714285714286
  • Resto da divisão: console.log(n1 % n2); Retorno: 3


2- Relacionais

Retornam valores booleanos (true e false)

  • Igual: console.log(n1==n2); Retorno: false
  • Diferente: console.log(n1!=n2); Retorno: true
  • Maior que: console.log(n1>n2); Retorno: true
  • Menor que: console.log(n1>n2); Retorno: false
  • Maior ou igual: console.log(n1>=n2); Retorno: true
  • Menor ou igual: console.log(n1<=n2); Retorno: false

OBS: "console.log é uma das formas de exibir. São elas:

  • console.log(msg): Exibe msg no console.
  • alert(msg): Exibe msg no popup.
  • document.write(): Exibe msg na página.


3- Lógicos

Vamos supor que uma variável "aprovado" esteja retornando um valor "true". O operador "!" inverte os valores. Vejamos:

  • console.log(aprovado); Retorno: true
  • console.log(!aprovado); Retorno: false


4- Unindo expressões

Nesses casos usamos E (&&) e OU (||), conforme a tabela verdade. Vejamos:

  • && (e): console.log(n1>=n2 && n1<=minimo); Retorno: false
  • || (ou): console.log(n1>=n2 || n1<=minimo); Retorno: true

Observe que, no primeiro caso, o retorno é "false". Isso porque no caso do E as duas expressões têm que ser verdadeiras para que a sentença toda seja true e uma delas não é (no caso, n1<=minimo). Já no segundo caso o retorno é "true" porque no caso do OU, apenas uma das expressões precisa ser verdadeira (no caso n1>=n2 é a verdadeira).

Unindo JS e HTML

Podemos fazer a ligação do JS com HTML buscando elementos do form HTML e enviando para o form JS e, em seguida, podemos exibir os resultados fazendo o caminho inverso. Abaixo segue um exemplo de codificação:

Código JS (dentro da tag <head></head>):

                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Document</title<
            
                <script>
                    function somar() { //criamos uma função que será retornada posteriormente

                        const n1 = Number(document.getElementById("n1").value); // Dentro da função, pegamos o valor 
                                                                                // dos elementos com as ids "n1"
                        const n2 = Number(document.getElementById("n2").value); // e "n2" transformados para "Number"
                        const soma = n1 + n2;
                                    
                        
                            /*exibindo o resultado -2 formas:
                            const resultado = document.getElementById("resultado") // var resultado é oq tiver no id=resultado
                            resultado.innerHTML = "O resultado é " + soma;         // muda o conteudo (innerHTL) do id=resultado*/
                        
                        document.getElementById("resultado").innerHTML = "O resultado é " + soma; //faz direto sem criar variaveis
                    }
                </script>
