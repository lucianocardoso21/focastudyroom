Importando e utilizando bibliotecas

Caso eu queira utilizar algum método eu posso utilizar bibliotecas. A maneira de fazer isso são duas:


1- Importar a biblioteca toda

Caso eu queira escolher nomes numa lista, por exemplo, podemos fazer da seguinte maneira: 

import random // importa toda a biblioteca "random" com todas as funções
            n1 = str(input(“insira nome 1: “))
            n2 = str(input(“insira nome 2: “))
            n3 = str(input(“insira nome 3: “))
            n4 = str(input(“insira nome 4: “))
            lista = [n1, n2, n3, n4]
            escolha = random.choice(lista)
            print(escolha)

2- Importar parte da biblioteca

Caso eu NÃO queira importar a biblioteca inteira, basta importar apenas os módulos que eu quero usar: Vejamos como Embaralhar uma lista: 

from random import shuffle // importar apenas a função "shuffle" da biblioteca "import"
            n1 = str(input(“insira nome 1: “))
            n2 = str(input(“insira nome 2: “))
            n3 = str(input(“insira nome 3: “))
            n4 = str(input(“insira nome 4: “))
            lista = [n1, n2, n3, n4]  // lista criada
            shuffle(lista)  // lista embaralhada
            print(lista)

Manipulação de strings

Vamos levar em consideração a seguinte frase: “Curso em Vídeo Python”

Em python, a string é armazenada cada letra individualmente. Ex: frase = "Curso de Python". O armazenamento dessa frase será C u r s o () d e () P y t h o n. Assim podemos fazer facilmente algumas operações.

Fatiamento

  • Frase[9] = Caractere na posição 9 da lista: V. Contando C como posição 0 e contando os espaços, a letra V na posição 9 será o 10 caractere.
  • Frase[9:13] = Víde: Pega da pos 9 até a pos 12 (exclui o 13 e pega 9, 10, 11, 12) Ultimo valor não entra na contagem.
  • Frase[9:21:2] = VdoPto. Pega da pos 9 até 20 (exclui 21), pulando de 2 em 2.
  • Frase[:5] = Curso. Não definindo parâmetro inicial, ele começa no 0 e vai até o indicado (no caso, pos4).
  • Frase[15:] =Python. Similar ao anterior, não indicando final, ele pega do indicado até o fim
  • Frase[9::3] =VePh. Pega do 9, vai até o final (não indicado), pulando de 3 em 3

Métodos

  • len(frase) = 21. len Retorna o comprimento da string
  • frase.count(‘o’) = 3. Conta quantas vezes aparece a ocorrência (letra o minúscula) na string
  • frase.count(‘o’,0,13): Já faz fatiamento. Conta os ‘o’s d 0 até 12.
  • frase.find(‘deo’) = 11. Na frase em questão, a sequencia ‘deo’ começa na pos11. Quando ele procura algo que não existe na string, o retorno é igual a -1.
  • ‘Curso’ in frase = true. Existe curso em frase.
  • frase.replace(‘Python’, ‘Android’). Substitui a palavra Python por Android.
  • frase.upper(): tudo maiúsculas
  • frase.lower(): tudo minúsculas
  • frase.capitalize() e frase.title(): tudo maiúsculas
