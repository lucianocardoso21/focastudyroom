Inserindo tags, elementos e atributos via Javascript

É possível criar tags e elementos html via Javascript, dinamicamente, bem como setar atributos. Vejamos o código a seguir: 

                
                    function verificar(){
                        var img = document.createElement('img');
                        var res = document.getElementById('res');
                        img.setAttribute('id','foto');
                        img.setAttribute('src','jova.jpg');
                        img.setAttribute('height', '100px');
                        res.appendChild(img);
                        }
No código acima, estamos inserindo Criando um elemento (tag) HTML, a seguir, Setamos a id como 'foto', depois Setamos a src (a imagem em si), que por sinal, ficou muito grande, então setamos a altura de 100px e aí vem o pulo do gato: Fizemos tudo isso aparecer dentro da variável 'res'

Verificando Radio Buttons

No caso de radio buttons, podemos verificar qual está marcado a depender do valor retornado. Vejamos o código abaixo: 


                <body>
                    <input type="radio" name="gen" id="masc" value="Masculino" checked>Masculino
                    <input type="radio" name="gen" id="fem" value="Feminino">Feminino 
 

                    <button type="submit" onclick="verificar()">Mandar
                    <div id="res1">div1
                    <div id="res2">div2
                    
                    <script>
                        function verificar(){
                        var genero = document.getElementsByName('gen');
                        var res1 = document.getElementById('res1');
                        var res2 = document.getElementById('res2');
                        res1.innerHTML = genero[0].checked;
                        res2.innerHTML = genero[1].checked;
                    }
                
                    </script>
                </body>
No código acima, estamos verificando qual radio button está marcado (checked) - que é diferente do outro checked no escopo html. Este checked está verificando qual radio button está marcado (se o da posição [0], [1] ou outra posição qualquer) e o outro está estabelecendo um dos campos como padrão (para evitar que os campos deixem de estar marcados.)
